Day 2: Basic Formulas & Cell References

Introduction to Formulas

All Excel formulas begin with an equals sign (=). Basic arithmetic operations:

Operation Symbol Example Result
Addition + =5+3 8
Subtraction - =10-4 6
Multiplication * =2*6 12
Division / =15/3 5
Exponentiation ^ =3^2 9
Percentage % =20% 0.2

Cell References

References allow formulas to dynamically update when cell values change:

Relative Reference: =A1+B1 (changes when copied to other cells)
Absolute Reference: =$A$1+$B$1 (stays fixed when copied)
Mixed Reference: =A$1+$B1 (row or column stays fixed)
Tip: Press F4 to cycle through reference types when editing a formula.

Basic Functions

SUM: =SUM(A1:A10) - Adds all numbers in the range
AVERAGE: =AVERAGE(B2:B20) - Calculates the average
MAX/MIN: =MAX(C1:C100) - Finds highest value in range

Day 22: Array Formulas (CTRL+SHIFT+ENTER)

Array Formula Fundamentals

Array formulas perform multiple calculations on one or more sets of values and then return either a single result or multiple results. They are entered by pressing Ctrl+Shift+Enter.

Examples of Single-Cell Array Formulas

Sum of Products: ={SUM(B2:B10*C2:C10)}

Multiplies corresponding values in two ranges and then sums the results.

Conditional Count: ={SUM(IF(B2:B10="Sales",1,0))}

Counts cells that meet a specific condition (equivalent to COUNTIF).

Examples of Multi-Cell Array Formulas

Transpose: ={TRANSPOSE(A1:A5)}

Converts a vertical range into a horizontal range.

List Unique Values: ={IFERROR(INDEX($A$2:$A$10, MATCH(0, COUNTIF($B$1:B1, $A$2:$A$10), 0)),"")}

Extracts a unique list of values from a range.

Tip: In newer versions of Excel, many array functions no longer require CSE (e.g., FILTER, UNIQUE, SORT).

Day 25: PivotTables - The Basics

What is a PivotTable?

A PivotTable is a powerful tool for summarizing, analyzing, exploring, and presenting data. It allows you to quickly transform rows into columns and vice-versa, and perform calculations on the data.

Creating a PivotTable

How to: Select your data → Insert tab → PivotTable

PivotTable Anatomy

The four main areas of a PivotTable:

  • Filters: Filter the entire PivotTable by one or more fields.
  • Columns: Display data horizontally.
  • Rows: Display data vertically.
  • Values: The data that is aggregated (sum, count, average, etc.).

A calculated column is a new column added to a table in the data model. Its values are calculated row-by-row.

Day 36: DAX - The CALCULATE Function

CALCULATE: The Heart of DAX

The CALCULATE function is the most important and powerful function in DAX. It modifies the filter context of a measure's calculation.

Syntax: CALCULATE(, , , ...)

Example: Sales for a specific product

=CALCULATE([Total Sales], 'Products'[Product Name]="Laptop")

This formula calculates the total sales, but only for the rows where the product name is "Laptop".

Example: Sales from the previous year

=CALCULATE([Total Sales], SAMEPERIODLASTYEAR('Dates'[Date]))

This formula calculates sales but shifts the date context to the same period in the previous year.

Day 40: Analyzing Trading Data & In-Depth DAX

Introduction to DAX (Data Analysis Expressions)

DAX is a formula language used in Power Pivot, Power BI, and SQL Server Analysis Services to create custom calculations in PivotTables and PivotCharts. It extends Excel's capabilities by allowing you to create measures, calculated columns, and tables.

Key DAX Concepts for Trading Data

  • Measures: Dynamic calculations that aggregate data, such as total profit or average trade duration. They are used in the Values area of a PivotTable.
  • Calculated Columns: Add new columns to a table, with a formula that is calculated row-by-row. For example, a "Profit/Loss" column based on entry and exit prices.
  • Evaluation Context: DAX has two contexts that affect how formulas are calculated:
    • Row Context: The row-by-row calculation for a calculated column.
    • Filter Context: The filters applied by the PivotTable (e.g., date, stock ticker, etc.).

Analyzing Trading Data with DAX

DAX Function Use Case for Trading Example Formula
SUM, AVERAGE, MAX, MIN Calculate basic metrics like total profit, average entry price, highest price reached, or lowest price hit. =SUM('Trades'[Profit])
CALCULATE The most powerful DAX function. It modifies the filter context to perform a calculation. =CALCULATE(SUM('Trades'[Profit]), 'Trades'[Result]="Win")
(Calculates total profit only for winning trades)
FILTER Returns a table filtered by a condition. Often used inside CALCULATE. =CALCULATE(AVERAGE('Trades'[Trade Duration]), FILTER('Trades', 'Trades'[Market]="Forex"))
COUNTROWS Counts the number of rows in a table. Useful for counting the number of trades. =COUNTROWS('Trades')

Day 49: Monte Carlo Simulation in Excel

What is a Monte Carlo Simulation?

A Monte Carlo simulation is a technique used to model the probability of different outcomes in a process that cannot be easily predicted due to random variables. It is often used in finance and risk analysis.

Building a Simulation Model

The basic steps to build a Monte Carlo model in Excel:

  1. Identify the uncertain variables (e.g., stock price, sales volume).
  2. Define the probability distribution for each variable (e.g., normal distribution).
  3. Use Excel's random number functions (RAND, RANDBETWEEN, NORM.INV) to simulate thousands of scenarios.
  4. Calculate the outcome for each scenario.
  5. Analyze the results to determine the probability of different outcomes.
Simulating a Normal Distribution: =NORM.INV(RAND(), mean, standard_dev)
